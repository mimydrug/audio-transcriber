## 📐 건축사 실무교육 (CPD) → [[건축사_실무교육_CPD]]

건축사 법정 교육 이수 이력 (`raw/study.md` 기준):

| 연도 | 주요 교육 주제 |
|------|----------------|
| 2014–2016 | 유지관리점검, 구조물 안전, 리모델링, ALC블록, 흙건축 |
| 2017 | BIM 기초~Advanced (1~3), 시각화 프레젠테이션 |
| 2019 | 에너지절약 녹색건축, 방수품질, 건축협정, 설계용역비 분쟁 |
| 2020 | 방수설계·기술, 해체공사 감리, 녹색건축인증(G-SEED) |
| 2021–2022 | 건축공사 감리실무 I·II, 교육시설안전인증, 그린스마트 미래학교 |
| 2023–2026 | 해체감리 보수교육, 정기점검 보수교육, 감리실무(콘크리트균열, 가설구조물), RC 공사감리 |

주요 반복 주제: **방수**, **감리**, **BIM**, **에너지·녹색건축**, **구조안전**

→ 연결: [[방수_습기제어]] · [[경량목구조]] · [[한국건축법규]]
