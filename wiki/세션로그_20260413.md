**교차분석 핵심 발견:**
- 자문이 제시한 **OCP 역설 논리** 검증 → 기술적으로 타당
- 피고 4대 주장 모두 오류 판정

**OCP 역설 구조:**
```
RTX 4070(200W) = PCIe 슬롯 75W + 케이블 125W
케이블 한계 = 75W → 과부하 166%
PSU OCP 임계치 = 216W (피고 스스로 제출)
125W < 216W → OCP 미작동 → 차단 없이 13개월 누적 소손
```

### 5. 기술 독립 검증 (3회 교차검증)

**생성 파일:** `사건_기술독립검증_202604.md`  
**방법:** 자문 완전 배제, 국내외 1차 자료만 활용

**검증 출처:**
- PCI-SIG 공식 규격 (PCIe 1.0~5.0)
- NVIDIA RTX 4070 공식 사양 및 Tom's Hardware 리뷰
- HP Support Community (공식 Z840 문서)
- Intel ATX 3.0 Multi-Rail PSU 설계 가이드
- Tom's Hardware OCP 전문 해설

**검증 결과 — 6개 명제 전부 사실 확인:**

| 명제 | 판정 |
|------|------|
| PCIe 슬롯 75W = PSU 무관한 물리 한계 | ✅ 사실 |
| 6핀 커넥터 75W = PCI-SIG 공식 규격 | ✅ 사실 |
| RTX 4070 = 슬롯 75W + 케이블 125W | ✅ 사실 |
| Z840 PSU 216W ≠ 커넥터 물리 한계 변경 | ✅ 사실 |
| OCP 216W 임계치 → 125W 과부하 미반응 | ✅ 사실 |
| 누적 열화 → 장기 소손 메커니즘 | ✅ 사실 |

**결론:** 자문 내용과 독립 검증 결과 완전 일치. 피고 주장 4개 전부 기술적 오류.

**HP 공식 커뮤니티 추가 발견:**  
"Do NOT use a 6-pin to 8-pin connector under any condition" (Z840에서 단일 6핀→8핀 젠더 절대 금지) → 피고 시공이 HP 자체 가이드라인에도 위반됨.

1. **가장 강력한 논점:** 피고가 스스로 제출한 "단일 레일 18A(216W)" 스펙이 OCP 미작동의 근거가 되어, 커넥터 소손의 인과관계를 과학적으로 입증함.

2. **HP 자체 가이드 위반:** HP 공식 커뮤니티가 Z840에서 단일 6핀→8핀 젠더를 명시적으로 금지함. 피고의 "워크스테이션이라 다르다" 주장이 오히려 HP 자체 기준에도 위반됨.

3. **을제12호증의 역이용:** 피고 제출 강원전자 3차 답변서에 "단일 6핀 + 젠더 = 위험"이라는 문구가 포함되어 있음 → 피고 증거가 원고 주장 입증.

4. **스티커 자기모순:** 4월 6일 서면("스티커를 원고가 봤어야") vs 4월 12일 서면("사전에 구두 설명했다") → 직접적 자기 모순.
