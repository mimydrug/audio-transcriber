## 📌 Conventions

- Internal links use `[[Page Name]]` syntax (double square brackets) — click to navigate or create the page.
- For large PDFs or documents: summarize via **NotebookLM** first, then save clean insights into `/wiki/` as structured Markdown.
- Source files always live in `/raw/` — never edit them here.
- This page is the **master index** — every new page should be linked from a section above.
- Dates in `YYYY-MM-DD` format throughout.
- **Active Now** block at the top should be updated whenever your current project or focus changes.
- **저장 우선순위:** /wiki/ 폴더가 기본 primary storage. Google Drive는 PDF 링크 필요 시 또는 명시적 요청 시에만 사용.
- **세션 로그:** 매 세션 종료/업데이트 요청 시 `세션로그_YYYYMMDD.md` 형식으로 저장.
