## 1. 인허가 종류 — 건축허가 vs 건축신고

### 건축신고 (Notification) 기준

| 조건 | 내용 |
|------|------|
| 관리지역·농림지역·자연환경보전지역 | 연면적 200㎡ 미만, 3층 미만 |
| 도시지역 | 연면적 85㎡ 이하 |
| 증·개축 | 증축 부분 연면적 85㎡ 이하 |

> **각 동 별도 판단 원칙 (유권해석 08-0051):** 같은 대지 내 여러 동의 경우 동별로 독립 판단. 합산 불필요.

### 사암리 적용 예시

| 동 | 행위 | 면적 | 인허가 |
|----|------|------|--------|
| 본채 | 증축 (발코니 실내화 + RC 확장) | 48.99㎡ | **건축신고** (증축부 기준) |
| 별채 | 신축 | 226.99㎡ | 검토 필요 (관리지역 200㎡ 초과) |

> 개발행위허가 의제: 건축신고 시 개발행위허가가 함께 의제 → 대지 전체 개발 규모(배수, 진입로)가 크면 지자체가 '허가' 절차 요구 가능

## 2. 시공자 자격 제한 — 건설산업기본법 제41조

### 핵심 조항

**「건설산업기본법」 제41조(건설공사 시공자의 제한) ①항**

> 다음 각 호에 해당하는 건축물의 건축 또는 대수선에 관한 건설공사는 **건설사업자(면허 소지자)**가 하여야 한다.
>
> 1. **연면적이 200㎡를 초과하는 건축물**
> 2. 연면적 200㎡ 이하인 건축물 중:
>    - 공동주택 / 다중주택 / 다가구주택
>    - 학교·병원 등 다중이용 건축물

### 증축 시 면적 산정

**「건축물의 시공자제한 업무처리요령」 제2조:**
> "기존 건축물을 증축 또는 대수선하는 경우 그 증축 또는 대수선하는 부분의 바닥면적의 합계로 한다."

**실무 리스크:**
- 증축 후 건축물 자체가 '연면적 200㎡ 초과 건축물' 지위 획득
- 지자체에 따라 증축 후 총면적 200㎡ 초과 시 종합건설업 시공 강제하는 경우 多
- 설계변경 시 증축+대수선 합산 200㎡ 초과 가능

> 사암리 본채: 증축 후 **208.25㎡** → 종합건설업 면허 시공 필수

### 위반 시 처벌 (제94조)

| 처벌 | 내용 |
|------|------|
| 형사처벌 | 5년 이하 징역 또는 5천만원 이하 벌금 |
| 행정처분 | **사용승인(준공) 거부 + 건축물대장 생성 불가** |

## 3. 건설사업자 (종합건설면허) 자격 요건

### '건설사업자'의 법적 정의

**「건설산업기본법」 제2조(정의) 제7호:**
> "건설사업자"란 이 법에 따라 **건설업 등록을 한 자**를 말한다.

### 왜 종합건설면허여야 하는가?

집을 짓는 것 = 기초·골조·마감·설비 등 여러 공종이 복합된 **'종합공사'** → 종합건설업(건축공사업) 필요.

**「건설산업기본법」 제9조 및 제13조:**
- 종합공사: 종합적인 계획·관리·조정을 하면서 시설물 설치 → **건축공사업**
- 전문공사: 시설물 일부 또는 전문 분야 → 실내건축공사업, 지붕판금공사업 등

### 건축공사업(종합건설업) 등록 기준

| 구분 | 요건 |
|------|------|
| **기술 인력** | 건설기술인 5인 이상 상시 고용 (건축기사 또는 중급 2인 포함) |
| **자본금** | 법인: 납입자본금 3.5억원 이상 / 개인: 자산평가액 7억원 이상 |
| **공제조합 출자금** | 자본금의 25~60% 현금 예치 |
| **사무실** | 독립된 사무 공간 (통신·사무설비 구비) |

## 4. 구조 안전 확인 요건

### 구조안전확인서 제출 대상 (건축법 시행령 제32조)

| 조건 | 요건 |
|------|------|
| 연면적 200㎡ 이상 | 구조안전확인서 + 구조기술사 날인 |
| 3층 이상 | 구조기술사 날인 |
| 특수 구조 (장스팬 등) | 구조계산서 첨부 |

**사암리 적용:**
- 본채 208.25㎡ → 구조안전확인서 필요
- I-Joist 7m 장스팬 → 구조 계산서 + 제조사 사양서 병행

## 5. 내화구조 기준 (건축법 시행령 제56조)

### 의무 대상

| 조건 | 내화구조 의무 |
|------|------------|
| 3층 이상 건축물 | 주요 구조부 내화구조 |
| 연면적 2,000㎡ 이상 | 주요 구조부 내화구조 |
| **2층 이하 단독주택** | **의무 대상 제외** |

> 사암리 본채(2층): 법적 내화 의무 없음. 단, H형강 노출 시 성능적 내화 권장.

### 성능적 내화 권장 기준

- 철골 임계온도: **538°C** (이 온도에서 강도 50% 이하 저하)
- 권장 내화 성능: **1시간**
- 내화 방법: 내화 뿜칠(Spray) / 내화 석고보드 2겹 피복

## 6. 단열 기준 — 건축물 에너지절약설계기준

### 지역별 단열 기준 (중부1지역 — 경기·인천 포함)

| 부위 | 중부1지역 열관류율 기준 (W/m²K) |
|------|-------------------------------|
| 외벽 | 0.170 이하 |
| 최하층 바닥 | 0.150 이하 |
| 최상층 천장·지붕 | 0.150 이하 |
| 창호 | 0.900 이하 (패시브: 0.8) |
| 출입문 | 1.400 이하 |

**PHIKO (한국패시브건축협회) 기준:**
- 외벽: 0.170 W/m²K (건에기 기준과 동일)
- 창호: 0.8 W/m²K (건에기보다 강화)
- 기밀: ACH 0.6 이하

## 7. 시공사 선정 시 확인 서류

| 서류 | 확인 내용 |
|------|---------|
| **건설업 등록증** | 업종 '건축공사업' 또는 '토목건축공사업' 확인 + 면허번호 |
| **KISCON 조회** | 키스콘(www.kiscon.net)에서 현재 영업정지·면허 유효 확인 |
| **국세/지방세 완납증명서** | 회사 재무 건실성 확인 |
| **건설공제조합 보증서** | 공사 이행 보증 |

> **경고:** 면허 없는 인테리어·소규모 업체와 직영 형태 계약 + 실제로 그들이 시공 = **불법 면허 대여** → 사용승인 거부

## 8. 개발행위허가 (관리지역)

### 국토계획법 적용

관리지역(계획관리지역) 내 건축 시:
- 건축신고와 함께 개발행위허가 의제 처리 (원칙)
- 단, 배수·진입로 등 대지 전체 개발 규모가 크면 지자체가 별도 허가 요구 가능

### 개발행위허가 심의 기준

| 항목 | 내용 |
|------|------|
| 경사도 | 평균 경사도 15° 이하 권장 |
| 개발면적 | 30,000㎡ 이하 (소규모) |
| 진입도로 | 차량 접근 가능 도로 확보 |
| 배수계획 | 우수 및 오수 배수 계획 제출 |
