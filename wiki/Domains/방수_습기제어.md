## 1. 방수의 핵심 원칙

> **습기는 항상 흐른다. 막는 것이 아니라 관리하는 것이다.**

두 가지 전략:
1. **방수 (Waterproofing):** 물의 침입 차단 — 외부 → 내부
2. **투습 (Vapour Permeable):** 내부 수분 배출 — 내부 → 외부

이 두 방향이 동시에 작동해야 벽체·지붕 부식이 없다.

## 2. 방수 공법 분류

### 2.1 시트방수 (Sheet Waterproofing)

| 구분 | 특징 | 적용 |
|------|------|------|
| APP 개량아스팔트 시트 | 열융착 방식, 신뢰성 높음 | 지붕, 테라스 하부 |
| TPO 시트 | 화이트 반사 코팅, 단열 효과 | 평지붕, 녹화 지붕 |
| EPDM 시트 | 신축성 뛰어남, EJ 구간 | EJ 가스켓, 특수 형상부 |

**시트방수 시공 순서:**
```
바탕 처리 (레이턴스 제거, 크랙 보수)
프라이머 도포
시트 붙이기 (열융착 또는 접착)
겹침: 가로 100mm, 세로 150mm 이상
단부 처리 (창호·벽 교차부 별도 상세)
```

### 2.2 도막방수 (Liquid Applied Waterproofing)

| 구분 | 특징 | 적용 |
|------|------|------|
| 우레탄 도막 | 신축성 우수, 형상 자유 | 욕실, 발코니 |
| 폴리우레아 | 빠른 경화, 내충격성 | 주차장 바닥 |
| 시멘트 방수 | 저렴, 내구성 보통 | 지하 벽체 |

**도막방수 두께:** 최소 2mm (욕실), 3~4mm (테라스·발코니)

## 3. 부위별 방수 상세

### 3.1 테라스·발코니 (RC 슬래브 상부)

**공법 선택: 페디스탈 공법 (Raised Floor System) 권장**

```
[상부 마감] 하드우드 데크 또는 포세린 타일
[페디스탈]  조절식 받침대 (배수 공간 확보)
            ↑ 이것이 역전방수(Inverted)와의 차이점
[방수층]    APP 개량아스팔트 2겹 또는 TPO 시트
[바탕]      RC 슬래브 (구배 1/50 이상)
```

**역전방수 (Inverted Insulation):**
```
[마감] 자갈 또는 페이버
[단열] XPS 압출법 단열재 (방수층 위에 설치 = 역전)
[방수] APP/TPO 시트
[구조] RC 슬래브
```
- 장점: 방수층이 온도 변화·UV로부터 보호됨
- 단점: 단열재 고정 어려움 (자중 또는 앵커)

> ⚠️ **EJ 구간 테라스 방수:** 이 부분 비용 절감 시 2층 테라스 하부에서 반드시 누수 하자 발생 (사암리 프로젝트 경고)

### 3.2 EJ 구간 방수 (이질 구조체 접합부)

이질 구조체(LGSF ↔ RC)가 만나는 EJ 구간은 별도 방수 처리 필수.

```
[외부] 알루미늄 커버 플레이트 (빗물 1차 차단)
[기밀] 가변형 투습방수 기밀테이프 (이중 시공)
[탄성] EPDM 신축 가스켓 (±15mm 변위 수용)
[단열] 연질 우레탄 폼 (탄성 유지형)
```

핵심: 기밀테이프는 **가변형(Variable Diffusion)** 제품 사용 — 계절에 따라 투습성이 변해 내부 습기 제어

### 3.3 발코니 하부 역단열

```
[발코니 바닥] 마감재
[단열] 발코니 하부 역단열 — 구조체 외부에 단열 배치
[구조] 캔틸레버 슬래브
[열교 차단] Schöck Isokorb 또는 동등 열교 차단재
```

열교 없이 발코니를 캔틸레버로 내밀 때 필수.

### 3.4 욕실 방수

```
[마감] 포세린 타일 (논슬립)
[접착] 탄성 타일 접착제
[방수] 우레탄 도막 2~3mm (코너·배수구 주변 보강)
[보호] 시멘트 보드 (CRC) 또는 방수 석고보드
[구조] 슬래브 또는 목재 바닥 (방습층 추가)
```

욕실 바닥 구배: 배수구 향해 1/50 이상

### 3.5 지붕 방수 — Cold Roof (경사지붕)

**Cold Roof 원리:**
- 단열재를 구조체(트러스·서까래) 안에 배치
- 단열재 위에 **통기층** 확보 후 지붕 마감재 설치
- 결과: 지붕 마감재가 외기 온도를 유지 → 적설 자연 미끄러짐, 결로 방지

```
[외부] 징크 거멀접기 마감
[통기] 통기층 (최소 40mm) — Cold Roof 핵심
[바탕] OSB 또는 합판
[방수] 투습방수지 (Tyvek Commercial 등)
[단열] 그라스울 또는 셀룰로오스 충진 (서까래 사이)
[기밀] 가변형 기밀막
[내부] 설비층 + 석고보드
```

### 3.6 천창 (Skylight) 방수 디테일

대형 천창(3,000×1,600) 징크 지붕 적용 시:

```
[천창 프레임] 업스탠드 최소 150mm 이상
[플래싱]     징크/리드 플래싱 + 실란트 이중 방수
[커브 처리]  업스탠드 커브 (역결로 방지)
[드레인]     업스탠드 하부 배수구 (2개소 이상)
```

역결로 방지: 천창 하부 내측에 결로수 집수 드레인 설치

## 4. 투습방수지 (Housewrap) 설치 원칙

1. **방향:** 투습(내→외) + 방수(외→내) 방향 확인
2. **겹침:** 수평 100mm 이상 (위 장이 아래 장 위에 겹침 = 비늘 방향)
3. **테이프:** 이음부 전용 접착 테이프로 기밀 처리
4. **창호 연결:** 창호 주변 물막이 테이프로 기밀 연속성 유지

---

## 5. 기밀층 (Airtight Layer)

### 가변형 기밀막 (Variable Diffusion Membrane)

| 제품 예시 | 특징 |
|---------|------|
| Intello Plus | 습도에 따라 투습성 변화 (dry: 방습 / wet: 투습) |
| DB+ | 표준 가변형 기밀막 |
| Pro Clima 제품군 | 독일 패시브 하우스 표준 |

**원리:**
- 겨울 (건조): 투습성 낮춤 → 내부 수분이 벽체로 들어가지 않음
- 여름 (습함): 투습성 높임 → 벽체 수분이 실내로 건조됨

---

## 6. 결로 방지 원칙

**이슬점(Dew Point) 관리:**
- 단열층을 외벽 외측에 충분히 배치 → 이슬점이 항상 단열재 내부에 위치하도록
- 내단열만으로는 이슬점이 구조체 안에 생김 → 결로 위험

**실내 상대습도:** 50~60% 유지 (HRV 환기 장치로 제어)
