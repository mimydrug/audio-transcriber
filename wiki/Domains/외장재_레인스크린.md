# 🧱 외장재 & 레인스크린 (External Cladding & Rain Screen)

> **도메인 지식 — 범용 참조용**
> **연결:** [[경량목구조]] · [[방수_습기제어]]
> **적용 프로젝트:** [[프로젝트_개요]]
> **Last updated:** 2026-04-14

---

## 핵심 원칙: 통기층(Rain Screen)은 협상 불가

목조주택 외장재 선택 시 디자인보다 선행되어야 할 **제1원칙**:

> **외벽 OSB 합판과 외장재 사이에 공기가 흐르는 통기층(Air Gap)이 없으면 100% 하자.**

목조주택은 습기 관리가 생명이다. 통기층이 없으면 합판 부식 → 구조체 손상 → 건물 수명 단축.

## 1. 외장재 비교 분석

| 구분 | 벽돌 타일 | 스터코 (EIFS) | 세라믹 사이딩 |
|------|----------|------------|------------|
| **물리적 안전성** | ⚠️ 위험 — 목재 수축·팽창으로 타일 탈락·크랙 높음 | 보통 — 드레인랩 없으면 벽체 부식 | ✅ 최우수 — 전용 클립으로 통기층 자동 확보 |
| **시공성** | 상 (습식, CRC 하지 필수) | 중 (습식, 날씨 영향) | 하 (건식, 품질 균일) |
| **유지보수 (LCC)** | 줄눈 오염, 백화 관리 | 3~5년 주기 재도장 | 10~15년 후 실리콘(코킹)만 교체 |
| **초기 비용** | 가장 비쌈 (Index: 180) | 가장 저렴 (Index: 100) | 중-고가 (Index: 150) |
| **권장** | RC 외단열에만 조건부 허용 | ❌ 목조에서 위험 | ✅ 목조에 최우선 추천 |

## 2. 세라믹 사이딩 (Ceramic Siding) — 상세

### 외벽 단면 구성 (외부 → 내부)

```
[마감]  16mm 세라믹 사이딩 (오토클린 코팅)
         ↑ 주의: 14mm(못 시공) 금지 → 크랙·오염 발생
[클립]  전용 금속 클립 → 15mm 통기층 자동 확보
[방수]  투습방수지 Tyvek HouseWrap
[구조]  11.1mm OSB 합판 + 2x6 구조목
```

### 핵심 포인트
- **16mm 클립 시공** = 통기층 자동 확보 + 크랙 방지
- 자가세정(Self-cleaning) 코팅 → 유지보수 최소화
- 건식 공법 → 공기 단축, 품질 균일
- 목조주택 수명 연장에 결정적

---

## 3. 벽돌 타일 (Brick Tile) — RC 외단열 한정

### 목조에 타일 직부착 = 금기

목조주택에 벽돌 타일 사용 시 **반드시 CRC 시멘트보드 하지** 작업 필요.

### 외벽 단면 구성 (외부 → 내부, RC 외단열 기준)

```
[마감]  파벽돌/점토타일 (탄성 접착제 + 줄눈)
[바탕]  12mm CRC 보드 (시멘트보드) + 메쉬 미장
[통기]  18mm 수직 방부목상 @406 (통기층 필수)
[단열]  외단열: EPS 비드법 2종 1호 180mm 이상
[방수]  투습방수지 (Tyvek Commercial)
[구조]  RC 벽체
```

### 주의사항
- 접착제: 반드시 **탄성 접착제** 사용 (일반 시멘트 모르타르 금지)
- 통기층: 방부목상 수직 설치 (배수구 하부 필수)
- 목조에 사용 시: 타일 탈락 리스크 → 10년 내 하자 확률 높음

## 4. 스터코 (Stucco / EIFS)

### 위험 경고

> **일반 타이벡 위에 스티로폼을 붙이면 집이 썩습니다.**

스터코에서 가장 중요한 것: **드레인랩(DrainWrap)** 사용.

### 외벽 단면 구성 (외부 → 내부)

```
[마감]  실리콘 페인트 (Stucco Flex 등 탄성 제품)
[단열]  50mm EPS + 메쉬 미장
[배수]  드레인랩 DrainWrap (주름진 투습방수지) ← 핵심
[구조]  11.1mm OSB 합판 + 2x6 구조목
```

### 리스크
- 습기 배출구(Drain) 없으면 벽체 부식
- 창호 주변 방수 디테일이 약하면 누수
- 눈물 자국 오염 발생 쉬움

## 5. 통기층 공법 비교

| 방식 | 통기층 확보 방법 | 적용 외장재 |
|------|---------------|-----------|
| 클립 방식 | 금속 클립 → 15mm 자동 확보 | 세라믹 사이딩 |
| 방부목상 | 18mm 수직 방부목상 @406 | 벽돌 타일, 일부 금속 패널 |
| 드레인랩 | 주름진 투습방수지 (DrainWrap) | 스터코/EIFS |

---

## 6. 투습방수지 (Housewrap) 종류

| 제품 | 용도 | 특징 |
|------|------|------|
| Tyvek HouseWrap | 일반 목조 외벽 | 투습·방수 표준 |
| Tyvek Commercial | 고성능·상업용 | 내구성 강화 |
| DrainWrap | 스터코 하지 | 주름 구조로 배수 기능 추가 |
